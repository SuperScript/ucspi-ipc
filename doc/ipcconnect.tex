\section{The \cmd{ipcconnect} program}

\subsection{Interface}
\begin{code}%
  ipcconnect \var{path}
\end{code}
where \cvar{path} is the filename for the client to connect to.

\cmd{ipcconnect} connects to a server at \cvar{path}.  It sends its
standard input to the server, and prints any data it receives from the
server to standard output.  \cmd{ipcconnect} can be used to transfer
binary data.  It does no conversion.
