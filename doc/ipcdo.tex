\section{The \cmd{ipccat} program}

\subsection{Interface}
\begin{code}%
  ipcdo \var{user} \var{prog}
\end{code}
where \cvar{user} is a user name and \cvar{prog} is one or more arguments
specifying a program to run via \href{\cmd{ipccommand}}{ipccommand.html}.

\cmd{ipcdo} uses \cmd{ipccommand} and \href{\cmd{ipcclient}}{ipcclient.html} to
connect to the \href{\cmd{ipcexec}}{ipcexec.html} server (specified at compile
time).  It sends any input it receives to the \cmd{ipcexec} server, and prints
any data it receives.
