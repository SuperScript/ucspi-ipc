\section{The \cmd{ipcrulescheck} program}

\subsection{Interface}
\begin{code}%
  ipcrulescheck \var{cdb}
\end{code}
where \cvar{cdb} is the name of the cdb-format file created with
\cmd{ipcrules}.  \cmd{ipcrulescheck} says what \cvar{cdb} will direct
\cmd{ipcserver} to do with a connection from a process with effective
user ID \cmd{\$IPCREMOTEEUID} and effective group ID
\cmd{\$IPCREMOTEEGID}.
