\section{The \cmd{ipccat} program}

\subsection{Interface}
\begin{code}%
  ipccat \var{path}
\end{code}
where \cvar{path} is the filename for the client to connect to.

\cmd{ipccat} connects to \cvar{path}.  It prints any data it receives.
\cmd{ipccat} can be used to transfer binary data.  It does no
conversion.
